# 示例 4：微博 / 即刻 短贴

场景预设：微博 / 即刻。模式：默认 + 轻度人味儿质感。

## 前

```
🌟 重磅推荐！

在 AI 写作越来越普及的今天，如何让你的文字保持人味儿，成为了一个非常
重要的话题。今天给大家分享一个超好用的工具 —— Anti-Vibe Writing！

✨ 它能帮你：
1️⃣ 一键去除 AI 痕迹
2️⃣ 让文字更有温度
3️⃣ 保留你独特的风格

干货满满，建议收藏！🔥
```

## 后

```
AI 写的稿子，问题不在写错。问题在"一看就是写出来的"。

Anti-Vibe Writing 就做最后这一步：删模板腔、删空话、把"赋能打通"换成
人话。中英文都行。

不重写观点，不加新内容。
```

## 改了什么

- 全部 emoji 删干净
- "重磅推荐 / 超好用 / 干货满满 / 建议收藏"全删
- 数字序号列表换成一段实话
- "在 AI 写作越来越普及的今天"这种开头删除
- 段落破碎度调整到适合手机阅读
- 末尾"不重写观点，不加新内容"，主动说"不做什么"反而比"能做什么"更可信

## 人味儿质感的痕迹

- "问题不在写错。问题在'一看就是写出来的'"拆成两个短句制造停顿，没用破折号（破折号现在是扎眼的 AI 标点，见 1.5.0）
- "中英文都行"短句单独成段
- 没有标准化的结构开头 / 结尾
