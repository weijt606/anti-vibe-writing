# 示例 7：中文推特 / X 发帖

场景预设：推特。模式：默认 + 轻度人味儿质感。

## 前

```
🚀 重磅分享！

在 AI 写作日益普及的今天，如何让文字保持人味儿、避免千篇一律，已经成为
每一位创作者都绕不开的重要话题。正因如此，我们精心打造了 Anti-Vibe
Writing —— 一款强大且易用的工具，致力于帮你将生硬的 AI 初稿，一键转化为
自然、地道、有温度的优质内容。

✨ 核心亮点：
1️⃣ 智能去除 AI 痕迹
2️⃣ 全面提升可读性
3️⃣ 完美保留你的独特风格

赶快试试吧，链接在评论区👇
```

## 后

```
AI 写的中文，毛病不在语法，在"一读就知道是机器写的"。

赋能、打通、不仅……而且、首先其次最后，这几个词一冒头，人味就没了。

Anti-Vibe Writing 只干一件事：把这些痕迹删掉，换成人话。不动你的观点。
```

## 改了什么

- emoji、序号 emoji（1️⃣2️⃣3️⃣）全删
- "重磅分享 / 精心打造 / 强大且易用 / 致力于 / 一键转化 / 优质内容"，整套营销腔删掉
- "在 AI 写作日益普及的今天"这种 AI 标配开头删掉，第一句直接给判断（1.6.0 命名为"套路化开头"）
- 三条"核心亮点"列表换成一句具体的话：它干的就是一件事
- 没有"赶快试试吧"，观点立住了，要不要点链接读者自己决定
- 字数从 130 多压到 60 出头，符合中文推特的真实节奏

## 人味儿质感的痕迹

- 第二句直接把"赋能、打通、不仅……而且、首先其次最后"几个词拎出来当反面教材，具体到词，比说"各种套话"可信
- "一冒头，人味就没了"是大白话，不是书面语
- "只干一件事""不动你的观点"用短句收尾，留下停顿

## 为什么这版更地道

- 第一句用"毛病不在 X，在 Y"的口语判断句，不用"问题在于"
- 通篇没有四字成语堆砌，没有欧化的"将……转化为……"
- 用"干一件事""换成人话""不动"这种动词，而不是"实现转化""保留风格"这种名词化表达
- 读出来像一个人在跟同行聊天，不像产品公告
