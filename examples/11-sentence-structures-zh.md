# 示例 11：中文套路化句式专项（1.6.0 新 pattern）

这条专门演示 1.6.0 新加的中文句式痕迹：套路化开头（"在这个 XX 的时代，我们每个人都……" / 说教式的"记住，真正重要的是……"）、"以前……现在……"时间对比、"总之 / 归根结底 / 说到底"收尾、鸡汤 / 口号升华、"什么都对但什么都没说"的圆滑结论。

这些骨架在删干净黑话之后还会留下：句子里一个套话词都没有，读起来照样像 AI。

场景：公众号 / 博客里一段"该不该用 AI 写作"。模式：默认（干净）。

## 前

```
在这个 AI 飞速发展的时代，我们每个人都无法回避一个问题：到底还要不要自己动笔？

以前，写一篇文章要查资料、列提纲、反复打磨，常常熬到深夜；现在，只要一句话，
AI 几秒钟就能给你一篇结构完整的初稿。这无疑是一场效率的革命。

但记住，真正重要的从来不是工具，而是使用工具的人。我们既要拥抱技术带来的便利，
也要警惕对它的过度依赖，做到具体问题具体分析。

总之，AI 是把双刃剑。归根结底，能不能写出打动人心的文字，取决于你有没有一颗
真诚的心。愿我们都能在这个时代，写出属于自己的声音。
```

## 后

```
AI 几秒钟就能给你一篇结构完整的初稿，这点确实比自己从头写快。

但初稿快不等于成稿好。AI 出的稿子读起来都像一个模子里倒出来的：套话、空泛、
什么都对又什么都没说。真正费时间的从来不是打字，是想清楚你到底要说什么、跟
别人有什么不一样。这部分 AI 替不了。

所以我的用法很简单：让 AI 出初稿打底，我自己花时间改观点和例子。它省的是打字
的体力，省不掉想清楚要说什么的那部分。
```

## 改了什么（一条 pattern 对一处修复）

- **套路化开头（"在这个 XX 的时代，我们每个人都……"）**：删，第一句直接说 AI 能干什么
- **"以前……现在……"时间对比**：不铺"以前熬夜、现在一句话"的对比骨架，压成一句"几秒钟就能给初稿，比从头写快"
- **夸饰（"一场效率的革命"）**：删
- **说教式开头（"记住，真正重要的是……"）**：删掉居高临下的口吻，把"想清楚要说什么"落成具体动作
- **伪平衡 / 圆滑结论（"既要……也要……、具体问题具体分析"）**：删，换成明确的立场（"我的用法很简单：……"）
- **收尾套话（"总之……归根结底……"）**：删，不用连接词宣布总结，句子讲完就停
- **鸡汤 / 口号升华（"愿我们都能……写出属于自己的声音"）**：删，结尾落到具体用法，不灌鸡汤

## 效果

前版套了六种结构骨架（时代开头、时间对比、说教、伪平衡、总结连接词、鸡汤收尾），通篇没有一个套话词，但零个具体做法，读完不知道作者到底主张什么。后版把骨架全拆了，给出一个能照着做的用法（AI 打底 + 自己改观点和例子），并下了判断："AI 替不了想清楚要说什么"。读起来像一个人讲自己怎么用 AI，不像公众号鸡汤。
