# 示例 2：中文产品介绍

## 前

```md
# 产品介绍

## ✨ 概述

本产品致力于为广大用户打造一站式的智能化解决方案，通过 AI 技术赋能业务，
让数据更智能、协作更高效、决策更精准。

## 🚀 核心特性

- 一站式数据接入
- 全链路智能分析
- 端到端业务闭环
```

## 后

```md
# 项目名

把散落在 Notion、飞书、邮件里的客户记录拉到一个地方，自动按公司聚合。
适合销售团队规模在 10 到 50 人之间、已经在多套工具里记客户的团队。
```

## 改了什么

- emoji 段头全删
- "致力于打造一站式智能化解决方案"换成可被验证的事实
- 三连排比"更智能、更高效、更精准"全删
- "赋能 / 打通 / 闭环"全删
- 加了"适合谁"，具体到团队规模，读者立刻能判断
