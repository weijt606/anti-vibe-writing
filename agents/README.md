# Local agent files

Place local agent configuration files in this folder.

The default file for this project is `anti-vibe-writing-dev.agent.md`.

That file is intentionally listed in `.gitignore`, so contributors can customize their local developer agent without changing the public repository.

Use `anti-vibe-writing-dev.agent.example.md` as the public template. Copy it to `anti-vibe-writing-dev.agent.md` for local use.
