# 中文前后对照基准

英文那套基准对中文不够用。中文 AI 文有自己的痕迹。这份是中文场景下的回归参考。

改 skill 之后，如果中文输出还像下面"前"的样子，就说明规则没生效。

## 基准 1：README 开头

**测试点：**
- 删掉"在……方面"开头
- 砍掉三连排比
- 把"赋能"换成具体动作

**前：**

```md
# 产品介绍

## ✨ 概述

本产品致力于为广大用户打造一站式的智能化解决方案，通过 AI 技术赋能业务，
让数据更智能、协作更高效、决策更精准。

## 🚀 核心特性

- 一站式数据接入
- 全链路智能分析
- 端到端业务闭环
```

**后：**

```md
# 项目名

把散落在 Notion、飞书、邮件里的客户记录拉到一个地方，自动按公司聚合。
适合销售团队规模在 10 到 50 人之间、已经在多套工具里记客户的团队。
```

**为什么这样改：**
- 砍掉所有 emoji 段头
- "致力于打造一站式智能化解决方案"换成可被验证的事实（拉数据 / 自动聚合）
- 三连排比删干净
- 加了"适合谁"，具体到团队规模，读者立刻能判断是否相关

## 基准 2：产品文档

**测试点：**
- 把"通过……来……"句式拆掉
- 把"即可"、"便可"还原成"就 / 只要"
- 删"首先 / 其次"段头

**前：**

```md
首先，用户需要进入设置页面。其次，通过点击"添加密钥"按钮，
在弹出的对话框中输入您的 API Key，从而完成密钥的配置。
配置成功后，系统将自动为您启用相关功能，您即可开始使用。
```

**后：**

```md
打开设置页，点"添加密钥"，把 API Key 粘进去。
保存之后下次启动就生效，不用再手动开开关。
```

**为什么这样改：**
- "首先 / 其次"删掉，靠句子顺序自己承担流程
- "通过……从而……"折成动作链
- "您"这种过度敬语换回中性的"你"或省略
- "即可"换成"就"或者完全不写

## 基准 3：创始人 / 产品笔记

**测试点：**
- 砍掉"我们致力于"开头
- 让段尾停在事实上，不要拔高到"行业未来"
- 替换互联网黑话

**前：**

```md
我们致力于通过 AI 技术为内容创作者赋能，打通从创作到分发的全链路闭环。
我们坚信，在生成式 AI 浪潮下，唯有不断深耕用户心智、持续打磨产品体验，
才能在这场变革中占据一席之地。这不仅是一次产品升级，更是我们对未来
内容生态的一次深度思考。
```

**后：**

```md
我们做的事很窄：给一篇已经写好的稿子做"去 AI 味"的最后一遍润色。
不是从头写、不是改观点、不重新组织结构，只把读起来像机器的地方
换成像人写的。这一步看起来小，但稿子能不能发出去经常卡在这一步。
```

**为什么这样改：**
- "致力于赋能"全删，换成对项目边界的清晰描述（做什么、不做什么）
- "全链路闭环"删掉
- 没有上升到"未来内容生态"，保留具体的功能范围
- 暖度来自"看起来小，但稿子能不能发出去经常卡在这一步"这句对读者处境的体察，不来自"我们坚信"

## 基准 4：技术备忘

**测试点：**
- 把过度对仗的总结拆成具体的取舍
- 删【背景】【现状】【建议】这种段头
- 让结论真的下判断，不要"具体问题具体分析"

**前：**

```md
## 【背景】

目前的数据管道在一定程度上能够满足业务需求。

## 【现状分析】

- 系统整体运行较为稳定
- 但仍存在一定的优化空间
- 未来或许可以进一步提升

## 【建议】

综合考虑各方面因素，建议采取较为均衡的策略，
具体问题具体分析，因时制宜，因事制宜。
```

**后：**

```md
现在的数据管道还能用，但有两个已知风险：上游断流没有兜底、
重试逻辑在并发高的时候会重复写库。两个都还没出过线上事故。

如果这季度不动，省两周工，继续承担每月一两次脏数据的清理成本。
如果现在动，多两周排期，之后这两类问题不用再回头处理。

我倾向现在动。但如果 Q2 排期已经满，可以接受拖到 Q3。
```

**为什么这样改：**
- 段头【背景】【现状】【建议】全删
- "一定程度上"、"或许"、"可能"几乎都换成具体描述
- "具体问题具体分析"换成真的分情况：什么情况下怎么选
- 最后一句下了倾向，但保留了让步条件，这才是真平衡，不是套话平衡

## 基准 5：落地页文案

**测试点：**
- 删"打造 / 助力 / 赋能"
- 把抽象收益换成可验证的承诺
- 不要 emoji，不要三连

**前：**

```md
我们为您打造极致丝滑的写作体验，助力您释放创作潜能，
让每一篇文章都更专业、更地道、更有温度。
```

**后：**

```md
把 AI 写的稿子放进来，删掉模板腔、删掉空话、把"赋能打通"换成人话。
不重写观点、不加新内容。
```

**为什么这样改：**
- "极致丝滑"、"释放潜能"全删
- 把承诺收窄到具体能做的事，没收益吹牛
- "不做什么"也写出来，这种克制反而比夸张更可信

## 退化警报

中文输出退化的几种典型样子，命中就要回头：

- 删掉了 emoji 但保留了【】段头
- 没了"赋能 / 打通"，但句尾还在用"……让 X 更 Y"的三连
- "首先 / 其次"删了，但每段开头换成"接下来 / 然后 / 此外"
- 中文里夹了一堆英文术语反而显得专业
- 把"我们致力于"换成"本产品致力于"，只是换了主语，套话还在
- 把整篇结构压扁成一段长流水账，丢掉了读者真正需要的小标题
- 删了黑话，但句子还是翻译腔（"被……所""作为一个……""对……进行……"照旧）
- 一句里仍堆着两三个四字成语（事半功倍、得心应手、一目了然）
- 念出来不像人说话，书面到没有一个气口
