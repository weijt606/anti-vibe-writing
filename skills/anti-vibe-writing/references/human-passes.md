# Human Passes

Use these passes in order for a full rewrite. After each pass, make a quick sanity check that the earlier gains still hold.

## Pass 1: Intent and Stance

**Focus:** Does the piece know what it wants to say?

**What to check:**
- The main point arrives too late.
- The opening clears its throat instead of making a claim.
- The piece explains the topic without taking a position on it.
- Necessary nuance is buried under unnecessary hedging.

**Common failures:**
- "This document aims to explore..."
- "There are several important considerations..."
- A soft preamble before the actual point appears

**Process:**
1. Mark the core claim and the intended reader outcome.
2. Cut setup until the point lands earlier.
3. Turn weak neutrality into a clear stance where the source already implies one.
4. Keep only the qualifiers that protect accuracy.

## Pass 2: Structure and Flow

**Focus:** Does the piece unfold in a natural reading order rather than an outline template?

**What to check:**
- Headings for every paragraph
- Lists where prose would read faster
- Summary before argument
- Symmetry that exists only because the model likes symmetry
- Sections that restate each other

**Common failures:**
- Intro, overview, key takeaways, conclusion all saying the same thing
- Equal-weight sections for points that do not matter equally
- Numbered frameworks without a real sequence

**Process:**
1. Collapse duplicate sections.
2. Merge thin subsections into fuller paragraphs.
3. Put the strongest material first.
4. Keep only the structure the reader will actually use.

## Pass 3: Concrete Language

**Focus:** Is the writing saying something real?

**What to check:**
- Abstractions standing in for actions
- Nominalizations instead of verbs
- Adjectives doing work that nouns and verbs should do
- Broad claims that could fit any product, team, or memo

**Common failures:**
- improve outcomes
- enhance alignment
- unlock value
- robust and scalable platform

**Process:**
1. Underline vague words and abstractions.
2. Replace them with concrete actors, actions, objects, and consequences.
3. Prefer plain verbs over inflated phrasing.
4. If a sentence cannot be made specific, cut it or shorten it.

## Pass 4: Rhythm and Temperature

**Focus:** Does the writing sound like a person with judgment rather than a machine with manners?

**What to check:**
- Every sentence landing at the same length and pressure
- Transition words propping up every paragraph
- Fake warmth or generic encouragement
- Endless caution that drains force from the claim
- Flat cadence with no compression or release

**Common failures:**
- "Let's dive in"
- "I hope this helps"
- "Thoughtfully designed"
- "We are excited to share"
- A paragraph that sounds polite but says nothing memorable

**Process:**
1. Read aloud or simulate the cadence mentally.
2. Shorten sentences that wander.
3. Keep some short sentences where emphasis helps.
4. Remove friendliness that exists only as filler.
5. Add warmth through concrete detail, measured confidence, and cleaner emphasis.

## Pass 5: Format and Markdown Cleanup

**Focus:** Is the formatting helping the reader, or just performing structure?

**What to check:**
- Emoji or icon headings
- Decorative bold labels
- Template sections such as Overview, Key Features, Benefits, or Conclusion
- Tables for ideas that do not need a table
- Nested bullets where a paragraph would be clearer
- Heavy markdown that makes the draft feel pre-fabricated

**Common failures:**
- "✨ Key Features"
- "🚀 Getting Started"
- Bolded labels at the start of every bullet
- FAQ sections with no real reader questions behind them

**Process:**
1. Remove decorative icons, emoji, and ornamental formatting.
2. Reduce the heading ladder.
3. Turn scaffolding into prose when the prose reads better.
4. Preserve markdown only where it improves scanability or reference value.

## Pass 6: Integrity Check

**Focus:** Is the rewrite sharper without becoming less true?

**What to check:**
- Claims that got stronger than the evidence allows
- Lost qualifiers that actually matter
- Changed numbers, names, links, or technical terms
- Tone improvements that accidentally altered the intended stance

**Process:**
1. Compare the rewrite against the source one more time.
2. Restore any nuance that was necessary for accuracy.
3. Confirm that the stronger voice still reflects the source.
4. Run the final checklist before returning the draft.

## Quick Path

For low-stakes cleanup, run only these steps:

1. Pass 1: Intent and stance
2. Pass 3: Concrete language
3. Pass 5: Format and markdown cleanup
