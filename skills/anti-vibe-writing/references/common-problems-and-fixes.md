# Common Problems and Fixes

Use this reference when the draft is bad in recognizable ways and you need a direct fix.

## Problem: AI Intro Runway

**Symptom:** The draft spends a paragraph preparing to make a point instead of making it.

**Fix:** Cut the runway. Start with the claim, the tension, or the practical point. Move setup later only if it still matters.

## Problem: Template README Shape

**Symptom:** The file defaults to Overview, Features, Benefits, Why It Matters, Conclusion, even when the content is thin.

**Fix:** Organize around what the reader actually needs: what this is, why it exists, how to use it, and what to watch for. Remove template headings that do not earn their place.

## Problem: Icon-Led Structure

**Symptom:** Section titles rely on emoji or icons to create energy: "✨ Features", "🚀 Quick Start", "💡 Tips".

**Fix:** Remove the decoration. If the section still matters, write a stronger heading. If not, fold it into the surrounding prose.

## Problem: Faux Warmth

**Symptom:** The draft sounds friendly because it keeps signaling friendliness rather than because it is useful or precise.

**Fix:** Replace soft filler with direct help. Warmth should come from clarity, respect for the reader's time, and concrete language.

## Problem: Consultant Veil

**Symptom:** The prose sounds polished but evasive: lots of frameworks, leverage, outcomes, and strategic alignment.

**Fix:** Name the actor, the action, and the consequence. Prefer verbs over nouns and plain English over process language.

## Problem: Balanced Blur

**Symptom:** Every sentence is softened twice, and no paragraph seems willing to decide what matters.

**Fix:** State the main claim plainly. Keep only the one caveat needed for truth. Carry nuance inside the sentence rather than around it.

## Problem: Markdown as Camouflage

**Symptom:** The draft looks organized because it has lots of headings, bullets, tables, and bold labels, but the thought itself is still generic.

**Fix:** Simplify the formatting until the actual writing has to carry the meaning. If the prose cannot stand without the scaffolding, the prose still needs work.

## Problem: Mirrored Ending

**Symptom:** The ending just replays the intro with slightly different words.

**Fix:** End on the strongest implication, next step, or final sentence of consequence. If no ending is needed, stop earlier.
