# Host Profile 模板

填这份模板的过程是学习模式的产物。每一项**必须配样本证据**，没有证据宁可留空，也不要编。

复制此文件，按用户提供的样本一项项填。最终的 profile 文件按 host 命名（如 `host-profile-zhangsan.md`），放在 `assets/` 下或外部仓库都行。
